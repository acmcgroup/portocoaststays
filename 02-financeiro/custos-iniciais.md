# Custos de Setup Inicial

Investimento necessário para colocar os 2 apartamentos operacionais como alojamento local.

---

## Setup por Apartamento

### Tecnologia de Acesso

| Item | Modelo sugerido | Custo estimado |
|------|----------------|---------------|
| Fechadura inteligente | Nuki Smart Lock 3.0 | 150–250 € |
| Teclado de acesso (keypad) | Nuki Keypad 2.0 | 80–100 € |
| Acesso porta do prédio (relé WiFi) | Nuki Opener ou solução DIY | 80–150 € |
| **Subtotal tecnologia** | | **310–500 €** |

### Equipamento do Apartamento

| Item | Notas | Custo estimado |
|------|-------|---------------|
| Máquina de lavar roupa | Se não existir | 300–500 € |
| Máquina de café | Alto impacto em reviews | 80–200 € |
| Smart TV | Se não existir | 250–400 € |
| Router WiFi rápido | Mínimo 100Mbps garantido | 50–100 € |
| **Subtotal equipamento** | | **680–1.200 €** |

### Roupa e Consumíveis (stock inicial)

|------|-----------|---------------|

| Almofadas e edredão qualidade hotel | — | 100–200 € |
| Kit boas-vindas inicial (café, açúcar, chá, sabão) | — | 30–50 € |
| **Subtotal roupa/consumíveis** | | **330–600 €** |

### Marketing & Listing

| Item | Notas | Custo estimado |
|------|-------|---------------|
| Fotografia profissional | Não negociar — impacto crítico | 150–300 € |
| Guia digital (criação) | PDF ou página web simples | 0–100 € |
| **Subtotal marketing inicial** | | **150–400 €** |

## Retorno do Investimento

| Métrica | Valor |
|---------|-------|
| Investimento total (realista) | ~3.000 € |
| Lucro mensal (base) | ~1.810 € |
| **Break-even** | **~1.7 meses** |

> Com o cenário otimista (120 €, 80% ocupação), o break-even ocorre em menos de 1 mês.

---

## Notas de Poupança

- Se os apartamentos já têm máquina de lavar e Smart TV, o setup cai para ~1.500–2.000 €
- O keypad Nuki pode ser partilhado (gere múltiplos códigos por unidade)
- Fotos profissionais são o único custo "não negociável" — o retorno é imediato no ranking
- Stock de roupa de cama pode ser comprado faseado (2 sets no início, 3º set após primeira receita)
