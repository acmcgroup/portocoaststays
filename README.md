# Homestay — Gestão de Alojamento Local

> Negócio de gestão de alojamento local com 2 apartamentos próprios (Porto), com potencial de escala via gestão para terceiros.

---

## Contexto

- **Imóveis:** 2 apartamentos (propriedade da família)
- **Localização:** Porto
- **Modelo:** Gestão própria (sem empresa externa) + escalabilidade via revenue share com terceiros
- **Preço médio:** ~100 €/noite
- **Ocupação estimada:** 70%
- **Plataformas:** Airbnb + Booking.com

---

## Quick-Start — Primeiros 30 Dias

### Semana 1 — Setup técnico
- [ ] Instalar fechaduras inteligentes (ex: Nuki) em ambos os apartamentos
- [ ] Configurar sistema de acesso porta do prédio (relé WiFi ou intercom)
- [ ] Contratar fotógrafo profissional para os dois espaços

### Semana 2 — Plataformas e listings
- [ ] Criar conta Airbnb + Booking.com
- [ ] Escolher e configurar PMS (recomendado: Hostaway ou Guesty)
- [ ] Configurar channel manager (sincronização de calendários)
- [ ] Configurar Pricelabs para pricing dinâmico

### Semana 3 — Operação
- [ ] Contratar equipa/pessoa de limpeza local (outsourcing)
- [ ] Criar guia digital do apartamento (PDF ou página web simples)
- [ ] Configurar mensagens automáticas (confirmação, pré-chegada, check-out)
- [ ] Definir checklist de limpeza + protocolo de fotos pós-limpeza

### Semana 4 — Primeira reserva
- [ ] Publicar listings com preço competitivo inicial (para primeiras reviews)
- [ ] Testar fluxo completo de check-in autónomo
- [ ] Validar que todos os códigos expiram na hora do check-out

## O Verdadeiro Diferencial

Não é o apartamento. É:

1. **Automação** — escala sem esforço proporcional
2. **Pricing dinâmico** — diferença entre 1.500 € e 2.500 €/mês
3. **Gestão de reviews** — motor orgânico de crescimento
4. **Velocidade de resposta** — ranking nas plataformas
5. **Relação com proprietários** — escala via gestão de terceiros (20–25%)
